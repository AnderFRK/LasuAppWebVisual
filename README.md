# CRUD con Predicciones usando React, Tailwind CSS, MySQL y TensorFlow.js

## Descripción

Este proyecto es una aplicación **CRUD (Create, Read, Update, Delete)** desarrollada con:

- **React** para la interfaz de usuario.
- **Tailwind CSS** para el diseño y estilos.
- **MySQL** como base de datos relacional.
- **Node.js + Express** como backend API.
- **TensorFlow.js** para el cálculo de predicciones.

La aplicación permite gestionar registros almacenados en MySQL y generar predicciones basadas en los datos existentes.
